Libelli

Libelli is a collection of C/C++ micro libraries. Each library implements a common tool that can be included into any project, and used with ease.

Getting Started

Libelli is primarily a collection of Github repositories that can easily be integrated. But there is also a tool that will handle the integration and downloading of the libraries.

Prerequisites

git is necessary for all of these instructions. For the automated tool libcurl is also required.
